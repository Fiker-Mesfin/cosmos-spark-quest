"# CosmosExplorer" 


**Cosmos Explore** is an interactive space exploration platform designed to make learning about the universe engaging and accessible. Discover planets, stars, black holes, galaxies, moons, and other celestial objects through detailed information, stunning imagery, educational videos, and AI-powered explanations.

---

Features

-  Explore planets, moons, stars, galaxies, nebulae, black holes, comets, asteroids, and more.
-  AI-powered summaries that explain complex space concepts in simple language.
-  High-quality images of celestial objects.
-  Educational YouTube videos related to each object.
-  Powerful search functionality.
-  Detailed facts including:
  - Formation
  - Size
  - Distance from Earth
  - Temperature
  - Mass
  - Composition
  - Interesting facts
  - Scientific discoveries
-  Beautiful animations using Framer Motion.
- Responsive design for desktop, tablet, and mobile.
- Fast navigation with TanStack Router.

## Installation

### Clone the repository

```bash
git clone https://github.com/yourusername/cosmos-explore.git
```

### Navigate into the project

```bash
cd cosmos-explore
```

### Install frontend dependencies

```bash
cd client
npm install
```

### Install backend dependencies

```bash
cd ../server
npm install
```

---

##  Running the Project

### Start the backend

```bash
cd server
npm run dev
```

### Start the frontend

```bash
cd client
npm run dev
```

Open:

```
http://localhost:5173
```
